# BuscaHashtag-Front


#### Requerimentos
-----------------
* Versão do node.js:

      NODE.JS >= 12.15.0

* Versão do npm:

      NPM >= 6.13.7

* Na pasta raiz do projeto execute o camando:

      npm install
      

#### Executando o front-end
-----------------
* No diretório raiz execute o seguinte comando:

      npm run serve
